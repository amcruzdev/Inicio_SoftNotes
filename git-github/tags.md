### Tags 
1. `git tag`<p></p>
    ### Parametros 
    `-a <"nombre"> <hash>`<p></p>
    `-m <"mensaje">`<p></p>
    `-d <nombre_tag>`<p></p>
    `git tag -a <version>`<p></p>
    `git tag -f -a <version> -m <message> HASH`<p></p>
    `git tag -l`<p></p>
    `git tag -d`
2. `git show-ref --tags`<p></p>

### Remote tags
`git push origin --tags`<p></p>
`git push origin: refs/tag/nombre-del-tag`<p></p>
